## 📌 Project Overview

This Customer Dashboard was built in Tableau to analyze customer behavior, order patterns, sales performance, and top customer contributions.

The dashboard provides interactive insights that help businesses understand customer engagement and make data-driven decisions.

## 📊 Dashboard Features

### 1. Customer Overview
- Total Customers KPI
- Total Orders KPI
- Sales per Customer KPI

### 2. Monthly Trend Analysis
- Customer trends by month
- Seasonal performance tracking

### 3. Customer Distribution Analysis
- Orders per Customer Distribution
- Customer ordering behavior insights

### 4. Top Customers Analysis
- Top-performing customers
- Sales contribution analysis
- Profit analysis

### 5. Interactive Filters
- Year
- Region
- Category
